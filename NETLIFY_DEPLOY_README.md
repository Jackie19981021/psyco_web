## 📋 部署步骤

### 方法 1: 拖拽部署（推荐）
1. 打开 [Netlify Drop](https://app.netlify.com/drop)
2. 将整个 `build` 文件夹直接拖拽到网页中
3. 等待上传完成，获得临时链接

### 方法 2: Netlify 控制台部署
1. 登录 [Netlify](https://netlify.com)
2. 点击 "Add new site" → "Deploy manually"
3. 拖拽 `build` 文件夹到上传区域
4. 部署完成后获得永久链接
